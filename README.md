## 🚀 Whow to run this project?

1.  **Fetch project**

    Use git to fetch project on you local machine.

    ```sh
    git clone git@github.com:lsarkisov/calc_expression.git
    ```

3.  **Start runing project.**

    Just fallow these commands:

    ```sh
    cd <calc_expression_dir>/
    make && make run
    ```
